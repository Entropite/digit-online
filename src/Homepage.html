The Digit Programming Language

A programming language for the numerically inclined


Examples

4.19 - Increments a user-inputted value and prints it out

6.41111111151962043 - Prints out all the natural numbers up to an inputted value

411111115.6.615204369 - Adds two positive numbers

411111115.6.625104369 - Adds two negative numbers

42262222711115.04363597359 - A truth-machine

8..19.7..9.4.5161420735964661283 - Calculates the times table of an inputted positive integer
